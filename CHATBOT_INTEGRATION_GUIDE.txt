## ğŸš€ **Automated Integration Script**

### **PowerShell Script to Add Chatbot to All Pages**

Save this as `add-chatbot.ps1`:

```powershell
# Add chatbot to all HTML pages in Proteomorphic project

$rootPages = @("index.html", "knowledge.html", "iot-monitoring.html", "admin.html")
$subfolderPages = @(
    "dashboard\index.html",
    "analysis\input.html",
    "auth\login.html",
    "auth\signup.html",
    "report\view.html"
)

$chatbotCodeRoot = @"
    <!-- Proteomorphic Chatbot Widget -->
    <link rel="stylesheet" href="css/chatbot.css">
    <script src="js/chatbot.js"></script>
</body>
"@

$chatbotCodeSubfolder = @"
    <!-- Proteomorphic Chatbot Widget -->
    <link rel="stylesheet" href="../css/chatbot.css">
    <script src="../js/chatbot.js"></script>
</body>
"@

# Add to root pages
foreach ($page in $rootPages) {
    $file = "c:\Users\lenovo\report\proteomorphic\$page"
    if (Test-Path $file) {
        $content = Get-Content $file -Raw
        if ($content -notmatch "chatbot\.js") {
            $content = $content -replace "</body>", $chatbotCodeRoot
            Set-Content $file $content
            Write-Host "âœ“ Added chatbot to $page" -ForegroundColor Green
        } else {
            Write-Host "â—‹ Chatbot already in $page" -ForegroundColor Yellow
        }
    }
}

# Add to subfolder pages
foreach ($page in $subfolderPages) {
    $file = "c:\Users\lenovo\report\proteomorphic\$page"
    if (Test-Path $file) {
        $content = Get-Content $file -Raw
        if ($content -notmatch "chatbot\.js") {
            $content = $content -replace "</body>", $chatbotCodeSubfolder
            Set-Content $file $content
            Write-Host "âœ“ Added chatbot to $page" -ForegroundColor Green
        } else {
            Write-Host "â—‹ Chatbot already in $page" -ForegroundColor Yellow
        }
    }
}

Write-Host "`nâœ… Chatbot integration complete!" -ForegroundColor Cyan
```

**To run**:

## â“ **Troubleshooting**

### **Chatbot Button Not Appearing**

```
Problem: Button doesn't show up
Solutions:
1. Check file paths are correct
2. Verify chatbot.css and chatbot.js exist
3. Check browser console for errors (F12)
4. Clear browser cache (Ctrl+F5)
5. Ensure z-index is high enough (9999)
```

### **Chatbot Not Responding**

```
Problem: Bot doesn't answer questions
Solutions:
1. Check browser console for JavaScript errors
2. Verify chatbot.js loaded correctly
3. Test with simple question like "hi"
4. Check network tab for blocked resources
```

### **Styling Issues**

```
Problem: Chatbot looks broken
Solutions:
1. Ensure variables.css is loaded first
2. Check for CSS conflicts
3. Verify chatbot.css loaded after variables.css
4. Inspect element to see applied styles
```
